Networking commands
    ip a > similar to ifconfig, newer command, colored result
    ip n > similar to arp -a > shows ip address and associated MAC address
    ip r > similar to route > shows routing table
    iwconfig > wireless connection info
    netstat > shows open ports

File Manipulation commands
    touch * > create a new file with name *
    cat * > read * file
    nano * > graphical interface to read and edit txt files
    grep * > search for * in files
    mousepad * > create a file and display it in GUI
    gedit * > GUI for text file editing

Kali commands
    sudo systemctl enable/disable * > service * will run on start up
